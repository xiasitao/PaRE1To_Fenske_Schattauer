This repository hosts the codebase of the seminar paper Potentials of Financial Market-based Industrial Clustering by Tobias Schattauer and Moritz Fenske.

The directory data hosts the preprocessing code.
The directory analysis hosts the clustering and analysis code.
The directory wikipedia hosts the constituent list web crawling code.

The raw data can be downloaded from
https://drive.google.com/drive/folders/1xg0VZrf3Z4oEvug8PwUOZbZEOBn6xXBb?usp=sharing
